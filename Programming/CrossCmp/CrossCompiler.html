CROSS_COMPILE

Below are the steps I used to install the cross compiler.

  1. Download the cross compiler:
    >> wget -c https://releases.linaro.org/14.04/components/toolchain/binaries/gcc-linaro-arm-linux-gnueabihf-4.8-2014.04_linux.tar.xz
  2. Unzip it:
    >> tar xf gcc-linaro-arm-linux-gnueabihf-4.8-2014.04_linux.tar.xz
  3. export the CC env variable, it will be used in all compilations:
    >> export CC=`pwd`/gcc-linaro-arm-linux-gnueabihf-4.8-2014.04_linux/bin/arm-linux-gnueabihf-
  4. To test it:
    >> ${CC}gcc --version
    This will print the version of the cross compiler.
  5. As an useful additional thing, you can set and export the CC and CROSS_COMPILE environment variables in .bashrc. So whenever you start bash the variables are automatically set. You can echo an warning that reads: "CC and CROSS_COMPILE are set to so and so...." which is echoed when bash starts up. This will remind the user of the custom environment variables.
